# Изгиб Питона — Змейка

## Описание
Реализация логики игры «Змейка» на Python + `pygame`. Проект содержит исполняемый файл `snake_game.py`, вспомогательные скрипты и (опционально) результаты OCR в `extracted_texts.txt`.

Игра:
- двигаться можно стрелками или WASD;
- змейка растёт при съедании яблока;
- проход через границы игрового поля (wrap-around);
- при самосъедении — сброс в начальное состояние.
